# Jia Wei Huang Resume

Public resume and portfolio page for Jia Wei Huang, based on the Start Bootstrap Resume theme and customized for a Traditional Chinese resume.

## Live site

<https://blackjw1212.github.io/resume/>

## Local development

```bash
npm install
npm start
```

## Maintenance notes

- Keep public contact details intentional; do not leave placeholder email addresses in the page.
- Keep dependency and build tooling current before making larger visual changes.
- This repository is a customized resume site, not an upstream theme mirror.
