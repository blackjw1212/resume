# Jia Wei Huang Resume

Public resume and portfolio page for Jia Wei Huang.

<https://blackjw1212.github.io/resume/>

## Local preview

Serve the repository with any static HTTP server, then open `index.html` through that server so `data/resume.json` can be loaded.

## Maintenance notes

- Keep public contact details intentional; do not add private phone numbers, addresses, personal identifiers, tokens, or internal system details.
- Review `PRIVACY.md` before publishing content changes.
- Keep resume content in `data/resume.json`; render page structure through `index.html`, `css/resume.css`, and `js/resume.js`.
- Do not add third-party runtime scripts, remote fonts, or package-managed frontend dependencies unless there is a documented security review.
