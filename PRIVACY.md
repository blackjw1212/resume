# Privacy Notes

This repository is intended to contain only public resume content.

## Do not commit

- Phone numbers, private email addresses, home addresses, birthdays, government IDs, or other personal identifiers.
- Access tokens, API keys, passwords, certificates, SSH keys, or environment files.
- Private resume drafts, unpublished documents, internal system names, or non-public workplace details.

## Safe public content

- Public name and role summary.
- Public GitHub profile link.
- Resume experience, education, skills, certifications, and portfolio descriptions that are intentionally public.

Before publishing changes, review `data/resume.json` and the rendered page for unintended personal or internal information.
